# 🎮 Gaming Zone

Welcome to Gaming Zone!

This is a modern gaming website created using only HTML and CSS. It has a clean black and neon blue design and works well on both mobile and desktop devices.

## ✨ Features

- Black & Neon Blue Theme
- Responsive Design
- Home Page
- Games Section
- About Section
- Contact Section
- Simple and Beginner-Friendly Code

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📁 Files

- index.html
- style.css
- README.md

## 🚀 How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in any web browser.
